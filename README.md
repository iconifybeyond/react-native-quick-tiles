# react-native-quick-tiles

> ⚠️ **Status: Alpha / Work in progress.** APIs may change. Not yet production-ready.

**Add and control Android Quick Settings tiles from React Native — built on the New Architecture.**

A React Native library that wraps Android's [Quick Settings Tile API](https://developer.android.com/develop/ui/views/quicksettings-tiles) (`TileService`), letting you expose a tile in the system Quick Settings panel and react to taps, update its label/icon/state, and request the user add your tile.

> 🚧 This repository is being scaffolded. Full library structure, native Kotlin implementation, example app, and documentation are on the way.

## Roadmap

- [ ] Declare and register a `TileService`
- [ ] Update tile label, subtitle, icon, and active/inactive state
- [ ] Listen for tile tap / add / remove events in JS
- [ ] Prompt the user to add the tile (Android 13+)
